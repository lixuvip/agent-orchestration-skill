# 轻量自动化

仅在任务需要跨越当前 turn 周期或延迟运行时加载本文件。

- 使用产品原生 automation 工具；不要用 shell 循环、heartbeat 文件、自定义租约或聊天轮询模拟调度。
- 创建前检查已有 automation；同一目标优先复用或更新，不重复创建。
- 在 automation prompt 中写明结果、范围、权限、验证、频率和停止条件。
- 每次运行保持小步：检查当前状态，执行一个安全且有价值的动作，验证并记录简短结果。
- 外部写入或发消息前比较最新有效状态；没有实质变化时安静 no-op。
- 周期运行不会自动获得 merge、push、deploy、publish、费用、破坏性动作、密钥或扩大范围权限。
- 完成或用户要求停止时，暂停、删除或归档 automation，并用原生工具确认清理。
- 当前任务的短期回访使用 heartbeat；持久项目推进使用定时 local/worktree automation。

交付时说明 automation 名称、目标、人类可读频率、权限边界、下次运行和清理状态；除非用户要求，不展示原始调度表达式。
