# Status Request Template

Use this when a role thread is silent, ambiguous, stale, or missing verification.

```text
Status request for <TASK_ID>
Coordinator thread ID: <COORDINATOR_THREAD_ID>
Role thread ID: <ROLE_THREAD_ID_OR_UNKNOWN>

Please reply with one current status and evidence. Reply with one of:
- IN_PROGRESS
- DONE
- DONE_WITH_CONCERNS
- BLOCKED
- NEEDS_CONTEXT

Required details:
- Current summary:
- Changed files / files inspected:
- Verification run:
- Risks / blockers:
- Estimated next step:
```
