# 升级报告模板

```text
项目 Autopilot 升级

目标 ID: <GOAL_ID>
自动化 ID: <AUTOMATION_ID>
工作区: <PATH_OR_REPOSITORY>
Status: BLOCKED | NEEDS_CONTEXT | DONE_WITH_CONCERNS

我为什么停止:
- <一到三条说明>

证据:
- <COMMAND_OR_CHECK>: <RESULT>
- <ISSUE_PR_THREAD_OR_BRANCH_STATE>

已尝试:
- <ACTION_1_AND_RESULT>
- <ACTION_2_AND_RESULT>

需要你决策:
- <QUESTION_OR_PERMISSION_NEEDED>

安全选项:
- <OPTION_1>
- <OPTION_2>
- <OPTION_3>

建议下一步:
- <ACTION>
```
