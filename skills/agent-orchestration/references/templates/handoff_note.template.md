# Handoff Note Template

协调者把一个角色的结果交给下一个角色时，使用本模板压缩上下文。

```text
Previous role: <ROLE_NAME>
Previous status: DONE | DONE_WITH_CONCERNS | BLOCKED | NEEDS_CONTEXT

What was completed:
- <SUMMARY_ITEM_1>
- <SUMMARY_ITEM_2>

Affected files / areas:
- <PATH_OR_AREA>: <WHY_IT_MATTERS>

Verification already run:
- <COMMAND_OR_CHECK>: PASS | FAIL | NOT RUN

Open risks:
- <RISK_OR_NONE>

What the next role should do:
- <NEXT_ACTION_1>
- <NEXT_ACTION_2>

What the next role must not do:
- <FORBIDDEN_ITEM_1>
- <FORBIDDEN_ITEM_2>
```
