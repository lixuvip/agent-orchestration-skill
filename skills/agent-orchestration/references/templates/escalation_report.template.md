```text
Status: BLOCKED | NEEDS_CONTEXT | DONE_WITH_CONCERNS

- <ONE_TO_THREE_BULLETS>

Evidence:
- <COMMAND_OR_CHECK>: <RESULT>
- <ISSUE_PR_THREAD_OR_BRANCH_STATE>

Attempts made:
- <ACTION_1_AND_RESULT>
- <ACTION_2_AND_RESULT>

Decision needed:
- <QUESTION_OR_PERMISSION_NEEDED>

Safe options:
- <OPTION_1>
- <OPTION_2>
- <OPTION_3>

Recommended next action:
- <ACTION>
```
